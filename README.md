## Sobre o projeto

![screenshot](.github/cover.png)

<br />
Esse CRUD gerencia o inventário de produtos e dados de clientes. Os dados são salvos no localstorage e podem ser editados e deletados.

Link demo: https://bestclothes.netlify.app
